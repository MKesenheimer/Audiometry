# Audiometry

## Usage:
Compile and run Audiometry:

```
make
./SDLDisplay -i /Users/kesenheimer/stdout -h 600 -w 800
```

## Requirements
```
sudo port install libsdl2 libsdl2_gfx libsdl2_image libsdl2_mixer libsdl2_ttf
```
